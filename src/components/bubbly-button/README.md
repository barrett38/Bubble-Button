# Project Name

## 📌 Description

A button that releases Bubbles every time it is clicked.

## 📖 Installation

```sh
# Clone the repository
git clone https://github.com/barrett38/Bubble-Button/tree/main/src/components/bubbly-button

# Install dependencies
npm install
```

## 🛠️ Usage

```sh
# Run the project
npm start
```
